# Ohjelmoinninperusteet
Pieni ohjelmistoprojekti kurssille Ohjelmoinninperusteet 2025

## Mikä sovellus on?
Arvaa luku on interaktiivinen tekstipohjainen peli jossa tietokone arpoo satunnaisen numeron 1-100 ja pelaajan tulee arvata se voittaakseen pelin

## Mitä peli tekee?
Peli arpoo satunnaisen numeron lukujen 1 ja 100 väliltä
Peli kysyy käyttäjältä toistuvasti arvauksen
Peli ohjaa käyttäjää vihjeillä "Liian iso" ja "Liian pieni"
Peli päättyy käyttäjän arvatessa oikean numeron
Lopuksi peli paljastaa arvatun numeron sekä yritysten määrän

## Pelin toteutus
Peli totetettiin Pythonilla hyödyntäen satunnaislukugeneraattoria, toistorakennetta ja ehtolauseita.
Ohjelma arpoo luvun 1-100 ja pyytää käyttäjää arvaamaan luvun.
Jokaisen arvauksen jälkeen ohjelma kertoo onko arvaus liian suuri vai liian pieni.
Kun arvaus on oikein, silmukka päättyy ja ohjelma kertoo oikean luvun sekä yritysten määrän

## Pelin käynnistys
- Varmista, että Python on asennettuna 
- Lataa reporisotion GitHubista koneellesi
- Avaa komentorivi projektin kansioon
- Suorita komento:
    python peli_arvaa_luku.py
